# AvaChat Monorepo

## Structure
- `apps/mobile`: React Native (Expo) Super App.
- `apps/api`: NestJS Backend (AI Intent Engine).
- `packages/contracts`: Solidity Smart Contracts (Hardhat).

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment**:
   Copy `.env.example` to `.env` and fill in the values.

3. **Run Dev**:
   ```bash
   npm run dev
   ```

## Key Features
- **Smart Accounts**: Non-custodial wallets for every user.
- **Chat Finance**: Send, Split, Save via natural language.
- **Token Launchpad**: Deploy tokens with bonding curves.
- **Mini-App Runtime**: Run 3rd party apps securely.
